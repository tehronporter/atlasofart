# Atlas of Art — Agent Rules

## Mission
Build only the MVP for Atlas of Art.
Keep the scope tight, elegant, and map-first.

## Build Rules
- Work only on the current phase
- Do not refactor unrelated files
- Do not add unnecessary dependencies
- Prefer simple, readable code
- Keep the map as the primary UI focus
- Use local seed data for MVP
- Do not introduce Supabase unless explicitly requested later
- Explain major file changes before making large edits
- After each task, list files changed
- After each task, provide the exact next terminal command

## MVP Features Only
- Mapbox world map
- Artwork markers
- Artwork detail drawer
- Timeline filtering
- Related works logic
- Responsive polish
- Vercel-ready deployment

## Do Not Add Yet
- Authentication
- Supabase backend
- AI chat
- Getty full ingestion
- Complex search
- Admin tooling
- Over-engineered architecture
